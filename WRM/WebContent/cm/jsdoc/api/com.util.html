                  (static) getJSON(value) → {Object}

                  Author:
                  • Inswave Systems

                  JSON Object로 변환해서 반환한다.

                  Example
                  // 유효하지 않은 JSON 문자열 일 경우
com.util.getJSON("");
// return 예시)	null

// 유효한 JSON 문자열
var json = '{"tbx_sPrjNm":"1","tbx_sPrtLv":"2","tbx_sReqLv":"3"}';
com.util.getJSON(json);
// return 예시)	{tbx_sPrjNm: "1", tbx_sPrtLv: "2", tbx_sReqLv: "3"}

                  (static) setInterval(func, options)

                  Author:
                  • Inswave Systems

                  실행할 함수를 함수를 setInterval로 등록하여 함수가 주기적으로 계속 실행되도록 한다.SPA모드에서 페이지 이동 시 이 함수로 등록한 타이머를 자동으로 제거한다.

                  Example
                  com.util.setInterval(
	function() {
		com.win.alert("처리가 완료 되었습니다");
	},
	{ caller : grd_data1, delay : 2000, key : "interval1" }
);
                  Parameters:
                  Name Type Description
                  func function

                  실행할 함수
                  options Object

                  options인자로는 아래와 같은 인자가 사용된다.

                  Properties
                  Name Type Description
                  key String

                  timer를 구별하기 위한 키값. 이 값이 지정되지 않은 경우 키값을 func.toString().slice(0,30)을 키값으로 사용한다.
                  delay Number

                  setInterval의 2번째 인자값. func함수가 delay시간 이후에 실행되도록 한다. 기본값은 1이다.
                  caller Object

                  func내에서 this값을 caller로 변경한다.
                  args Object

                  func에 전달할 인자값. array형태로 인자를 전달한다.
                  before_call function

                  : func 함수가 실행되기 직전에 실행할 함수. func함수와 마찬가지로 data를 인자로 받는다.
                  callback function

                  : func함수가 실행된 후에 실행할 함수. func함수의 return값을 인자로 받는다

                  (static) setTimeout(func, options)

                  Author:
                  • Inswave Systems

                  실행할 함수를 함수를 setInterval로 등록하여 함수가 주기적으로 계속 실행되도록 한다.SPA모드에서 페이지 이동 시 이 함수로 등록한 타이머를 자동으로 제거한다.

                  Example
                  com.util.setTimeout(
	function() {
		com.win.alert("5분이 지났습니다.");
	},
	{ delay : 300000, key : "loginTimeout" }
);
                  Parameters:
                  Name Type Description
                  func function

                  실행할 함수
                  options Object

                  options인자로는 아래와 같은 인자가 사용된다.

                  Properties
                  Name Type Description
                  key String

                  timeout을 구별하기 위한 키값. 이 값이 지정되지 않은 경우 키값을 func.toString().slice(0,30)을 키값으로 사용한다.
                  delay Number

                  func로 지정한 함수가 실행 될 간격으로 기본값은 1ms(millisecond / 1000분의 1초)이다. javascript setTimeout의 2번째 인자값.
                  caller Object

                  func로 지정한 함수내에서 this값으로 지정 할 객체(웹스퀘어 컴포넌트 포함).
                  args Object

                  func에 전달할 인자값. array형태로 인자를 전달한다.
                  before_call function

                  : func로 지정한 함수가 실행되기 직전에 실행 할 함수. func함수와 마찬가지로 data를 인자로 받는다.
                  callback function

                  : func로 지정한 함수가 실행된 후에 실행 할 함수. func로 지정한 함수의 return값을 인자로 받는다.
